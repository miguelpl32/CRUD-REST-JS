+ Validar formulario

## Documentación
+ Iniciar proyecto se debe tener instalado NodeJS
+ Instalar el paquete JSON-SERVER (npm i -g json-server)
+ Para iniciarlo ejecutar en consola json-server db.json -p 4000

## Tecnología
+ JavaScript
+ Nodejs
+ Tailwind
